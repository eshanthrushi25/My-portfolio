# Personal Portfolio Website

A modern, responsive, and lightweight personal portfolio website designed to showcase a student's digital identity, technical skills, projects, and achievements. Fully deployed and accessible worldwide via GitHub Pages.

 **[Live Project Link] https://eshanthrushi25.github.io/My-portfolio/**

---

##  Tech Stack & Architecture

This project is built purely with native front-end web technologies to ensure exceptional performance, clean semantic layout structure, and modular maintenance.

*   **HTML5**: Leverages semantic tags (`<header>`, `<section>`, `<nav>`, `<footer>`) to maximize accessibility and SEO performance.
*   **CSS3 Layout Engine**: Employs **CSS Flexbox** and **Grid** models for clean visual hierarchy, alignment, and custom typography variables.
*   **Vanilla JavaScript**: Drives native client-side interactivity, performance optimization, and custom component events.

---

##  Key Features & Component Breakdown

*   **Responsive Navigation Menu**: A sticky desktop header that seamlessly transitions into a responsive **JavaScript-powered hamburger toggle** on smaller mobile breakpoints.
*   **About Me & Biography**: Dedicated section introducing professional objectives alongside visual profile placeholders.
*   **Education Timeline**: Structured sequential layout documenting academic background and coursework milestones.
*   **Interactive Projects Showcase**: A modern layout grid populated with project cards featuring explicit descriptions and active outbound demo links.
*   **Skills & Certifications Inventory**: Visually distinct blocks categorized by technical proficiencies, frameworks, and verified certifications.
*   **Validating Contact Interface**: A custom user-facing form integrated with visual social link assets for GitHub, LinkedIn, and email access.

---

## 📐 UI/UX Optimization & Mobile Responsiveness

*   **Fluid Media Queries**: Tailored CSS media breakpoints ensure absolute visual adaptability across ultra-wide monitors, standard laptops, tablets, and smartphones.
*   **Accessible Typography**: High-contrast font weights and color schemes applied to comply with standard readability guidelines.
*   **Fluid Interactivity**: Smooth scrolling mechanics and subtle UI hover state animations implemented to enrich user engagement.

---

## 📂 File Architecture

The repository maintains a clean, flat architecture for seamless compilation and server hosting:

```text
My-portfolio/
│
├── index.html     # Application skeleton and semantic structure
├── styles.css     # Global layout styles, grid engines, and typography
└── script.js     # Behavioral scripts and mobile menu toggle events
```

Local Deployment Setup

To clone and modify this repository locally on your machine, follow these instructions:

1. Clone the repository to your local directory:
   ```bash
   git clone https://github.com
   ```
2. Navigate directly into the project root directory:
   ```bash
   cd My-portfolio
   ```
3. Open the entry file directly in any modern web browser:
   ```bash
   open index.html
   ```
